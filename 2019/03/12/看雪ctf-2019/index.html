神秘来信

IDA反编译, 然后看算法, 输入6个字符, 第4, 5, 6位分别为3, 5, 3

前三位加起来等于149

然后一个循环

v3 = input[0] + 16 0 - 48
v3 = input[1] + 16 v3 - 48
…
v3 = input[5] + 16 * v3 - 48

动态调试以后发现会把输入的6个字符变成一组hex, 然后看到地址0x401353处有点可疑, 尝试输入, 就成功了…有点迷, 挖个坑有时间研究一下
