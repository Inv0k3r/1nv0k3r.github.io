babyrop

输入一个字符串和一个随机串通过strcmp比较, 如果比较成功可以到达漏洞点进行栈溢出

可以通过在输入串开头加'\x00'来绕过, 然后栈溢出leak libc基地址, 之后利用ropgadget搞个ropchain即可getshell

book manager

提供一个图书编辑功能, 首先输入图书名, 之后是以下功能

  • 添加chapter
  • 添加section
  • 添加text
  • 删除chapter
  • 删除section
  • 删除text
  • 书籍预览
  • 修改(章节/小节/文本)

程序启动后先malloc一个0x80的chunk, 然后清空前0x1E个字节

输入0x1e个字节的name, 复制到这个chunk前面, 有截断

  • 添加chapter

最多可以创建12个chapter, 从第32个字节开始判断, 满足12个以内就可以用来存放一个0x8的指针, 这个指针指向一个新创建的0x80的chunk, 前0x20字节用来存放chapter name

  • 添加section

最多创建10个section, 先判断输入的chapter name是否存在, 如果存在就在chapter的第32个字节处开始存放0x30的chunk的指针, 输出这个指针指向的地址, 之后输入一个section name, 存放在前32字节, 并在40字节处存放数字32

  • 添加text

输入一个section name, 遍历所有chapter来查找, 找到后, 在其第32字节处创建一个小于256的chunk, 然后存入输入内容

  • 删除chapter

输入chapter name, 找到后, 按照text->section->chapter的顺序来free, 然后置零

  • 删除section

输入section name, 先free text, 然后free section, 但是section指针未置零, 存在uaf

  • 删除text

输入section name, 只free text并置零

  • 预览

按顺序输出每个chapter和其section以及text

  • 修改

先输入要修改chapter/section/text, 然后输入各种name
